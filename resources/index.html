Beginner Friendly

These programs can be learned quickly and are easy to use, but they are limited in what they can do.

  • Game Maker Studio
  • RPG Maker (Often goes on sale on Humble Bundle)

Story Focused

These programs are good for creating narratives and story based games without too much coding.

  • TWINE
  • Ren'Py Visual Novel Engine

Fully-featured game engines

These game engines are versatile enough to make it possible to implement a large variety of game mechanics without having to do any low-level programming.

  • Unity 3D
  • Unreal Engine
  • CryEngine
